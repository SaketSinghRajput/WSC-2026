# State-Level Labs (Q1-Q8)

## Overview
Beginner-friendly labs mapped to the eight State-Level practice questions. Each lab uses a single VPC with basic networking and Free Tier defaults in us-east-1.

## Difficulty and Time
- Difficulty: Beginner
- Total time: ~4-5 hours for all eight labs
- Region: us-east-1 only

## Prerequisites
- AWS account with Free Tier eligibility
- IAM access to EC2, VPC, S3, RDS, ElastiCache, ELB
- AWS CLI configured (optional but recommended)
- SSH client installed

## Lab Listing
- Q1: VPC & CIDR Planning (public subnet focus)
- Q2: EC2 SSH Connectivity (key pairs, SG rules)
- Q3: S3 Static Website Hosting (public site)
- Q4: RDS MySQL Backup & Restore (snapshots)
- Q5: ALB Basics with Single Target (health checks)
- Q6: ElastiCache Redis Performance Test (basic CRUD)
- Q7: EC2 User Data Web Server (bootstrap)
- Q8: Cost Optimization & Cleanup (tear-down patterns)

## Recommended Order
Q1 → Q2 → Q7 → Q3 → Q5 → Q6 → Q4 → Q8

## Notes
- All labs include screenshot placeholders and verification checklists.
- Clean up immediately after each lab to avoid charges.
