# State++ Level Labs (Q1-Q7)

## Overview
Intermediate labs for seven State++ practice questions. Focus on multi-tier VPC design, NAT Gateway, multi-AZ patterns, and exam-speed execution in us-east-1.

## Difficulty and Time
- Difficulty: Intermediate
- Total time: ~7-10 hours for all seven labs
- Region: us-east-1 only

## Prerequisites
- Completion of State-Level labs (Q1-Q8)
- Comfort with VPC routing, NAT Gateway, and ALB basics
- AWS CLI configured; SSH client available

## Lab Listing
- Q1: Secure Multi-Tier VPC with NAT Gateway
- Q2: ALB with Two EC2 Instances Across AZs
- Q3: RDS Backup & Restore – Snapshots vs PITR
- Q4: Path-Based Routing with ALB
- Q5: ElastiCache Redis Failover Testing
- Q6: Cost-Optimized Multi-Tier Architecture
- Q7: Bastion-Only Access to Private Subnet

## Recommended Order
Q1 → Q7 → Q2 → Q4 → Q3 → Q5 → Q6

## Notes
- Labs emphasize verification steps and troubleshooting under time limits.
- Delete NAT Gateways and ALBs immediately after verification to control costs.
